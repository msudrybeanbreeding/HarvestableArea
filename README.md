For raw polygons containing data that can be opened in QGIS, please look into the "Shapefiles with Data" folder.
Examples of harvestable-area centroids can be found in the centroids and polygon folder: however, due to GitHub filesize only examples below 25mb have been uploaded.
Examples of analysis code can be found in the Analysis code section, which contain the code utilized to generate the figures. Additionally, SAS code can be found in this folder that was used to evaluate coefficients of variation, levels of significance and more.
